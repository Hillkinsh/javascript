2. flex-grow 分配剩余空间

基础空间和剩余空间

基础空间：width和flex-basis占用的空间。

剩余空间：总空间 - 基础空间

flex-basis: 10%; /*子项的占用宽，设置了会覆盖width/height*/

.item1 { flex: unset; width: 100px; }

.item2 { flex-grow:2; width: 20px; flex-basis: 140px; }

.item3 { flex-grow: 1; width: 80px;}

基础空间： 100 + 140 + 80 = 320; 剩余空间 180

剩余空间分为3份，每份60，item2 两份； item3 1份

所以他们的渲染宽度为：item2 = 140 + 60* 2 = 260px

item3 = 80 + 60 = 140px

3. flex-shrink 吸收超出空间

.item1 { flex: unset; width: 100px; }

.item2 { flex-shrink:1; width: 20px; flex-basis: 140px; }

.item3 { flex-shrink: 2; width: 80px;}

基础空间： 100 + 140 + 80 = 320; 超出空间 320 - 250 = 70

超出空间分为4份，item1未设置，默认1份； item2 1份； item3 2份

item1 分担 100 * 1 / （100 * 1 + 140 * 1 + 80 * 2）* 70 = 17.5； item1最终宽度：100 - 17.5 = 82.5

item2 分担 140 * 1 / （100 * 1 + 140 * 1 + 80 * 2）* 70 = 24.5； item1最终宽度：140 - 24.5 = 115.5

item3 分担 80 * 2 / （100 * 1 + 140 * 1 + 80 * 2）* 70 = 28； item1最终宽度：80 - 28 = 52
